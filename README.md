# Recreational-Sport-League

## How to install

- Clone or download the project.
- Open the solution in JetBrains Rider (or another C# IDE).
- Set WebApp as the startup project.
- Build and run the solution.

## How to use
1. Open the **Seasons** tab and create a season.
2. Open the **Teams** tab and create teams.
3. Open the **Referees** tab and create referees.
4. Open the **Team Standings** tab and add teams to the standings table.
5. Open the **Matches** tab and create matches.
6. On the **Matches** page, click **Edit** next to a match to enter match statistics.
   - Standings are updated automatically based on match results and fair play (cards).
